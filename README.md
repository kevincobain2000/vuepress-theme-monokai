## Installation

> Note: Work For VuePress 1.x

```shell
$ npm install vuepress-theme-monokai --save-dev
# or
$ yarn add vuepress-theme-monokai --dev
```

## Usage

Modify your `.vuepress/config.js` and provide the `theme` option.

```js
module.exports = {
  theme: 'monokai',
}
```
